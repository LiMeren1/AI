 Sales Forecasting with ML and Genetic Programming

## Опис проєкту
Міні-проєкт прогнозує щоденні продажі онлайн-магазину. Мета - допомогти бізнесу планувати запаси, промо-кампанії та навантаження персоналу.  

Побудовано та порівняно:
- **Базові лінії:** Naive та Seasonal Naive  
- **ML-модель:** RandomForest  
- **Генетичне програмування (GP):** SymbolicRegressor (gplearn)
